9.1 Overview

As part of this activity, you will download three âmessyâ data sets and use tidyverse functions to transform them into âtidyâ data. If you want to read more about tidy data, I suggest the following article for an introduction to tidy data: https://tidyr.tidyverse.org/articles/tidy-data.html.

The tidyr package provides functions for making messy data âtidyâ. For a complete list of the included functions, see https://tidyr.tidyverse.org/reference/index.html.

Understanding tidy data and how to reformat your data into a tidy format is very important, as many of the functions/packages that we will use this semester are written to work with tidy-format data. For example, ggplot (the R graphics package that we will be using) assumes tidy data. Taking the time now to get comfortable with what it means to have tidy data may save you substantial time later on.
